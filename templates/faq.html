What is this?

Okay, fair question. btray is a tool that helps you test out Braintree Webhooks! It acts as a catcher for webhook settings you create in your Braintree Sandbox.

How do I get started?

Really simple:

  1. In Braintree Gateway:
    1. Set up a Webhooks-Only role in your Sandbox gateway (uncheck every permission)
    2. Create a new user that has the Webhooks-Only role
    3. Get the merchant id, public key, and private key from the API Keys for that user
  2. In btray:
    1. Create a webhook config with the API credentials you grabbed from the gateway. (Name/note it something memorable)
    2. Copy the endpoint url for your webhook config.
  3. In Braintree Gateway:
    1. Create a new webhook setup that points to your endpoint url from btray
    2. Generate some webhooks! Click the test url to make sure you get a 200 OK, then generate subscription events or whatever else you want.
  4. Go look at your pretty webhooks in btray!
